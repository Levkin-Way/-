Design Tokens

Set up your design language for a consistent look across your project
Go to the Design Tokens Panel to see the available text styles, color and layout tokens
To see the Design Tokens panel you need to click outside of the canvas
Select the button to begin styling it from the Visual Inspector
Try changing the padding and background color using tokens
Notice that some inputs support color tokens, others layout tokens and some don't support tokens at all
Go to the Text Style section in the Inspector and link the desired text style token by clicking the icon
From here you can also create new text styles with the currently set properties if you press +
